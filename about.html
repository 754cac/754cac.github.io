Hello, I'm Steven

I'm a Computer Science student at PolyU with a background in cybersecurity and competitive CTFs. I am focusing my work toward DevOps and infrastructure: system administration, network engineering fundamentals, incident response, and cloud operations.

My practical skills include Linux system administration, scripting with Python and shell, Java for software work, and familiarity with SQL and networking concepts. I apply a security-minded approach to automation and reliable infrastructure rather than building large-scale cloud systems from scratch.

CTF competitions have strengthened my problem-solving under pressure, particularly in forensics and web/security challenges. Professional roles at Varmeego (Junior Programmer) and Code Free Soft (Software Tester) helped build production-quality development and QA experience.

I pursue continuous learning through certifications (Advent of Cyber, HackerRank) and hands-on labs for incident handling, automation, and cloud fundamentals. I'm actively developing skills in infrastructure automation and operational tooling.

Beyond technical work, I enjoy mentoring peers and translating telemetry into actionable insights. I'm seeking early-career roles in system administration, network engineering, incident response, or cloud operations where I can grow practical operational skills.
